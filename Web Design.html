Website Design: Adair Conlin

To create this website, I used a mix of Adobe XD and Adobe Illustrator. In Adobe Illustrator, I created a style-scape for my website so I would have a general idea of the colors and design that I wanted. This style-scape, shown below, includes my personal digital art as the inspiration for the color palette of my website

In Adobe XD, I planned out the visuals of the website for each page. During the process of building my website I made a few advanced changes, such as creating an animated slideshow for my digital portraits instead of having three still images. These visual can be seen below as well.

I built this website using HTML, CSS, JavaScript, jQuery, and Bootstrap 4.
